skeuo13 Alt Icons II 1.0
com.matruka.skeuo-alticons2
iOS supported: 10.0 to 13.3.1

Description

  • This is the second pack with alternative icons. It is recommended to activate them next to the Main Theme to be able to switch between different icons.
  • If you have any ideas or suggestions for improvement, please feel free to contact me. If you want to support me, then I would be happy about a small donation.
  • For best compatibility it is recommended to use SnowBoard by SparkDev. No additional SnowBoard Extensions need to be installed for this package to work.
