skeuo13 Page Dots 1.0
com.matruka.skeuo-pagedots
iOS supported: 10.0 to 13.3.1

Description

  • To enjoy the full skeuo13 experience, it is recommended to install this package.
  • For best compatibility it is recommended to use SnowBoard by SparkDev. In order for the Page Dots to be displayed, the current version of the SnowBoard Page Dots Extension must be downloaded and activated in the Extension preferences panel in the settings app.
